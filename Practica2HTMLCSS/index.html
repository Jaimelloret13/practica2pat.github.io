Bienvenidos a la pagina web oficial del Valencia CF

Este sitio web es una demostración de las capacidades de HTML y CSS aplicadas al club de mi corazon.

La página principal incluye un menú de navegación en la parte superior, para saber un poco de la historia del club, entradas y las últimas noticias del equipo. Existe secciones para mostrar varios tipos de contenido, como imágenes y tablas.
